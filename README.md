# Hangman game

## Project overview
This project was created in ReactJs framework. Hangman image is dynamically drawn in html canvas.Used sweet alert library to display customized alert modal.

## Available Scripts
Once you git clone current project remember to run `npm install` to install all dependencies.

#### Run project
In the project directory, you can run:

### `npm start`

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## CI/CD 
Project is using netlify service for CI/CD.

Open [LIVE DEMO](https://serene-curie-66f858.netlify.app/) to view it in the browser.
